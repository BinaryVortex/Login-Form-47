# Login Form

A clean and responsive login form built with HTML and CSS.

![Login Form Screenshot](./Screenshot%202024-07-11%20150557.png)

## Description

This repository contains a simple, modern login form layout created using plain HTML and CSS. It's designed as a lightweight UI component that can be used in personal projects, practice exercises, or as a starting point for more complex login interfaces.

## Features

- Responsive layout
- Simple, modern design
- Easy to customize (colors, fonts, background)

## Preview

See the screenshot above for a visual preview.

## Usage

1. Clone the repository:

   git clone https://github.com/BinaryVortex/Login-Form-47.git

2. Open `index.html` in your browser:

   - Double-click `index.html`, or
   - Serve the folder with a static server (optional):
     - Python 3: `python -m http.server 8000`
     - Node: `npx serve`

3. Customize `style.css` to change colors, spacing, or fonts.

## File Structure

- `index.html` - The markup for the login form
- `style.css` - Styling for the form and page
- `background6.jpg` - Background image used by the page
- `Screenshot 2024-07-11 150557.png` - Project screenshot

## Customize

- To change the background, replace `background6.jpg` with your image and adjust `background` properties in `style.css`.
- To change fonts, add a Google Fonts import in `index.html` and update the CSS.
